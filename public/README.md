## Logo
Please add the MADWESS logo image as `madwess-logo.png` in this directory.

The logo should be:
- PNG format
- Transparent background
- Recommended size: 512x512px or larger
- The logo features yellow "M" and blue "W" letters with "MADWESS" text below

Current logo reference: Yellow and blue interlocking M and W letters on black background with gray "MADWESS" text.
